Pengantar Uji Kruskal-Wallis

Uji Kruskal-Wallis merupakan alternatif non-parametrik untuk ANOVA satu arah (One-Way ANOVA) ketika asumsi normalitas data tidak terpenuhi. Uji ini membandingkan median dari tiga atau lebih kelompok independen.

Asumsi Uji:

  1. Data terdiri dari $k$ sampel acak independen dengan ukuran sampel masing-masing $n_1, n_2, \dots, n_k$.
  2. Variabel dependen diukur minimal pada skala ordinal (kontinu lebih disukai).
  3. Distribusi populasi di bawah hipotesis nol adalah identik.

Hipotesis:

$$H_0: M_1 = M_2 = \dots = M_k \text{ (Semua populasi memiliki lokasi median yang sama)}$$

$$H_1: \text{Minimal ada satu kelompok yang memiliki lokasi median berbeda}$$

Statistik Uji:

Rumus uncorrected (tanpa ties):

$$H = \frac{12}{N(N+1)} \sum_{j=1}^k \frac{R_j^2}{n_j} - 3(N+1)$$

Dimana $R_j$ adalah jumlah peringkat kelompok ke-$j$, $n_j$ adalah ukuran sampel kelompok ke-$j$, dan $N = \sum n_j$ adalah total semua observasi.

Koreksi adanya Ties (Angka Kembar):

Koreksi Ties penting karena jika terdapat angka kembar, standard error akan cenderung terlalu besar, membuat nilai uji statistik mengecil. Koreksi dilakukan dengan membagi $H$ dengan faktor koreksi $C$:

$$C = 1 - \frac{\sum (t_i^3 - t_i)}{N^3 - N}$$

$$H_c = \frac{H}{C}$$

Dimana $t_i$ adalah jumlah data yang kembar pada kelompok ke-$i$.

Kaidah Keputusan:

Tolak $H_0$ jika $H_c \ge \chi^2_{(1-\alpha, k-1)}$.

Uji Lanjut (Post-Hoc Tests)

Jika uji Kruskal-Wallis menghasilkan keputusan Tolak $H_0$, maka dilanjutkan dengan membandingkan pasangan kelompok untuk melihat pasangan mana yang berbeda signifikan.

1. Uji Dunn (Dunn's Test)

Menggunakan distribusi normal standar $Z$.

$$SE_{ij} = \sqrt{\left[ \frac{N(N+1)}{12} - \frac{\sum(t_i^3-t_i)}{12(N-1)} \right] \left(\frac{1}{n_i} + \frac{1}{n_j}\right)}$$

$$Z_{ij} = \frac{|\bar{R}_i - \bar{R}_j|}{SE_{ij}}$$

Dimana $\bar{R}_i = R_i / n_i$.

2. Uji Conover-Iman

Uji pasca-analisis varians peringkat yang lebih sensitif (berdasarkan distribusi t-Student dengan $df = N - k$).

$$s^2 = \frac{1}{N-1} \left[ \sum R_{ij}^2 - \frac{N(N+1)^2}{4} \right]$$

$$SE_{ij} = \sqrt{ s^2 \left( \frac{N-1-H_c}{N-k} \right) \left( \frac{1}{n_i} + \frac{1}{n_j} \right) }$$

$$t_{ij} = \frac{|\bar{R}_i - \bar{R}_j|}{SE_{ij}}$$

Koreksi Bonferroni: Jika melakukan $m = \frac{k(k-1)}{2}$ perbandingan berpasangan, taraf signifikansi disesuaikan menjadi $\alpha' = \alpha / m$ untuk mengontrol tingkat kesalahan tipe I.

Implementasi Kode Program

Berikut adalah cara menjalankan uji Kruskal-Wallis beserta uji lanjut Dunn dan Conover-Iman menggunakan R dan Python.

Bahasa R 
# 1. Masukkan data
I  <- c(83, 91, 94, 89, 89, 96, 91, 92, 90)
II <- c(91, 90, 81, 83, 84, 83, 88, 91, 89, 84)
III <- c(101, 100, 91, 93, 96, 95, 94)
IV <- c(78, 82, 81, 77, 79, 81, 80, 81)

# Buat data frame (format panjang)
nilai <- c(I, II, III, IV)
metode <- factor(c(rep("I", length(I)), rep("II", length(II)), rep("III", length(III)), rep("IV", length(IV))))
data_jagung <- data.frame(nilai, metode)

# 2. Uji Kruskal-Wallis
kruskal.test(nilai ~ metode, data = data_jagung)

# 3. Uji Lanjut (memerlukan package PMCMRplus)
install.packages("PMCMRplus")
library(PMCMRplus)

# Dunn Test (dengan penyesuaian p-value Bonferroni)
kwAllPairsDunnTest(nilai ~ metode, data = data_jagung, p.adjust.method = "bonferroni")

# Conover-Iman Test (dengan penyesuaian p-value Bonferroni)
kwAllPairsConoverTest(nilai ~ metode, data = data_jagung, p.adjust.method = "bonferroni")
Python (scipy & scikit-posthocs) 
import scipy.stats as stats
import scikit_posthocs as sp

# 1. Masukkan data
g1 = [83, 91, 94, 89, 89, 96, 91, 92, 90]
g2 = [91, 90, 81, 83, 84, 83, 88, 91, 89, 84]
g3 = [101, 100, 91, 93, 96, 95, 94]
g4 = [78, 82, 81, 77, 79, 81, 80, 81]

# 2. Uji Kruskal-Wallis
h_stat, p_val = stats.kruskal(g1, g2, g3, g4)
print(f"H-statistic: {h_stat:.4f}, p-value: {p_val:.4f}")

# 3. Uji Lanjut Dunn & Conover-Iman
data = [g1, g2, g3, g4]

# Dunn's Post-Hoc
dunn_matrix = sp.posthoc_dunn(data, p_adjust='bonferroni')
print("Dunn Matrix p-values:\n", dunn_matrix)

# Conover-Iman Post-Hoc
conover_matrix = sp.posthoc_conover(data, p_adjust='bonferroni')
print("Conover-Iman Matrix p-values:\n", conover_matrix)

Pengantar Uji McNemar

Uji McNemar adalah uji non-parametrik yang digunakan pada data berpasangan (paired data) berskala nominal/kategori dikotomus (biner: Ya/Tidak, Sukses/Gagal, Sebelum/Sesudah). Fokus uji ini adalah mendeteksi adanya perubahan akibat suatu perlakuan.

Tabel Kontingensi 2x2:

Sebelum \ Sesudah Ya (1) Tidak (0)
Ya (1) $a$ (Sama) $b$ (Berubah)
Tidak (0) $c$ (Berubah) $d$ (Sama)

Sel $b$ dan $c$ disebut sel diskoran (discordant cells) karena mewakili subjek yang mengalami perubahan pilihan. Sel $a$ dan $d$ adalah sel konkoran (concordant cells) yang tidak menunjukkan perubahan.

Hipotesis (Dua Sisi):

$$H_0: P_b = P_c \text{ atau } P_1 = P_2 \text{ (Tidak ada perubahan proporsi / perlakuan tidak efektif)}$$

$$H_1: P_b \neq P_c \text{ atau } P_1 \neq P_2 \text{ (Terdapat perubahan proporsi / perlakuan efektif)}$$

Statistik Uji:

  • Sampel Besar ($b + c > 25$): Menggunakan pendekatan distribusi Chi-Square dengan $df = 1$.

    Tanpa koreksi kontinuitas:

    $$T_1 = \frac{(b - c)^2}{b + c}$$

    Dengan Koreksi Yates (untuk mengkompensasi pendekatan kontinu pada data diskrit):

    $$T_{corr} = \frac{(|b - c| - 1)^2}{b + c}$$

  • Sampel Kecil ($b + c \le 25$): Menggunakan pengujian eksak berbasis distribusi Binomial dengan $n = b+c$ dan $p = 0.5$.

    $$T_2 = b \quad (\text{dimana } b \sim \text{Binomial}(b+c, 0.5))$$

Kaidah Keputusan:

Tolak $H_0$ jika $T_{hitung} \ge \chi^2_{(1-\alpha, 1)}$ (untuk sampel besar, $\chi^2_{0.95, 1} = 3.841$).

Python (statsmodels) 
from statsmodels.stats.contingency_tables import mcnemar

# 1. Matriks kontingensi
table = [[22, 24],
         [18, 15]]

# 2. Uji McNemar dengan koreksi Yates (default)
result_with_correction = mcnemar(table, exact=False, correction=True)
print(f"Chi2 (Yates): {result_with_correction.statistic:.4f}, p-value: {result_with_correction.pvalue:.4f}")

# 3. Uji McNemar eksak (binomial, jika b+c <= 25)
result_exact = mcnemar(table, exact=True)
print(f"p-value (Exact): {result_exact.pvalue:.4f}")

Pengantar Uji Cochran Q

Uji Cochran Q merupakan perluasan dari uji McNemar untuk membandingkan 3 atau lebih kondisi/perlakuan berhubungan ($k > 2$) pada subjek/blok yang sama. Data respon harus berskala biner ($0$ atau $1$). Uji ini fokus pada mendeteksi perubahan proporsi sukses di antara perlakuan.

Asumsi Uji:

  1. Subjek dipilih secara acak dari suatu populasi (blok acak).
  2. Setiap subjek diukur berulang kali di bawah kondisi yang berbeda (pengamatan berkorelasi).
  3. Skala pengukuran adalah nominal dikotomus (biner, e.g., $0 = \text{gagal}$, $1 = \text{sukses}$).

Hipotesis:

$$H_0: P_1 = P_2 = \dots = P_c \text{ (Semua perlakuan memiliki proporsi sukses yang sama/sama-sama efektif)}$$

$$H_1: \text{Minimal terdapat satu perlakuan yang memiliki proporsi sukses berbeda}$$

Statistik Uji:

$$Q = \frac{c(c-1) \sum_{j=1}^c C_j^2 - (c-1) N^2}{c N - \sum_{i=1}^r R_i^2}$$

Dimana:

  • $c$: Jumlah perlakuan/kolom
  • $r$: Jumlah blok/subjek/baris
  • $C_j$: Total nilai $1$ pada kolom (perlakuan) ke-$j$
  • $R_i$: Total nilai $1$ pada baris (blok) ke-$i$
  • $N$: Total keseluruhan nilai $1$ dalam tabel ($N = \sum C_j = \sum R_i$)

Kaidah Keputusan:

Tolak $H_0$ jika $Q \ge \chi^2_{(1-\alpha, c-1)}$.

Pengantar Uji Friedman

Uji Friedman merupakan alternatif non-parametrik untuk ANOVA dua arah berulang (Two-Way Repeated Measures ANOVA) ketika asumsi normalitas tidak terpenuhi. Uji ini mengevaluasi apakah ada perbedaan yang signifikan antara efek beberapa perlakuan ($k \ge 3$) pada beberapa blok ($b$) sampel berhubungan.

Asumsi Uji:

  1. Rancangan penelitian adalah rancangan blok lengkap acak (RCBD), terdiri dari $b$ blok independen berukuran $k$.
  2. Pengamatan di dalam setiap blok dirangking dari terkecil ($1$) hingga terbesar ($k$).
  3. Efek perlakuan konsisten dan tidak berinteraksi dengan efek blok.

Hipotesis:

$$H_0: M_1 = M_2 = \dots = M_k \text{ (Efek semua perlakuan adalah identik)}$$

$$H_1: \text{Minimal ada satu perlakuan yang memiliki efek berbeda}$$

Statistik Uji:

$$T = \chi_r^2 = \frac{12}{bk(k+1)} \sum_{j=1}^k R_j^2 - 3b(k+1)$$

Atau dalam format kuadrat deviasi (dari UAS 2025 Nomor 1):

$$T = \frac{12}{bk(k+1)} \sum_{j=1}^k \left[ R_j - \frac{b(k+1)}{2} \right]^2$$

Dimana $b$ adalah jumlah blok (baris), $k$ adalah jumlah perlakuan (kolom), dan $R_j$ adalah jumlah peringkat kolom ke-$j$.

Kaidah Keputusan:

  • Untuk sampel besar ($b > 15$ atau $k > 4$): Tolak $H_0$ jika $T \ge \chi^2_{(1-\alpha, k-1)}$.
  • Untuk sampel kecil: Gunakan tabel statistik kritis khusus Friedman.

Uji Lanjut Friedman (Post-Hoc)

Jika uji Friedman menolak $H_0$, uji perbandingan berpasangan (pairwise comparisons) dapat dilakukan dengan membandingkan selisih absolut peringkat kolom dengan batas perbedaan kritis (Critical Difference):

$$CD = z_{(1 - \alpha/c)} \sqrt{\frac{bk(k+1)}{6}}$$

Dimana $c = \frac{k(k-1)}{2}$ adalah jumlah perbandingan, dan $z$ adalah nilai tabel Z dengan taraf signifikansi terkoreksi Bonferroni.

Tolak kesamaan perlakuan $i$ dan $j$ jika $|R_i - R_j| \ge CD$.

Pengantar Uji Durbin

Uji Durbin digunakan untuk menganalisis rancangan blok tidak lengkap seimbang (Balanced Incomplete Block Design / BIBD). Uji ini digunakan jika kapasitas suatu blok terbatas ($k \lt t$), sehingga tidak memungkinkan menerapkan seluruh perlakuan ($t$) ke setiap blok ($b$).

Karakteristik BIBD:

  • $t$ (Treatments): Jumlah perlakuan yang ingin dibandingkan (misal: 7 jenis ban mobil).
  • $b$ (Blocks): Jumlah blok atau kelompok penguji (misal: 7 orang pengemudi).
  • $k$ (Block Size): Ukuran blok, yaitu batas kapasitas perlakuan yang dapat dicoba di setiap blok ($k \lt t$, misal: 1 mobil hanya muat 4 ban).
  • $r$ (Replications): Jumlah pengulangan untuk setiap perlakuan di seluruh blok (misal: tiap jenis ban diuji sebanyak 4 kali).
  • $\lambda$ (Lambda): Frekuensi kemunculan bersama setiap pasangan perlakuan dalam blok yang sama.
💡 Apa itu $\lambda$ (Lambda) secara mendalam?

Dalam rancangan acak kelompok lengkap (seperti Uji Friedman), setiap perlakuan diuji bersama di setiap blok. Namun, pada Uji Durbin (BIBD), kapasitas blok terbatas ($k \lt t$). Kita tidak bisa memasukkan semua perlakuan ke satu blok.

Agar perbandingan tetap seimbang (Balanced), setiap pasangan perlakuan (misalnya perlakuan A & B, B & C, dst.) harus dibandingkan secara adil. Oleh karena itu, setiap pasang perlakuan harus muncul bersama-sama di dalam blok yang sama sebanyak $\lambda$ kali.

Nilai $\lambda$ ini dihitung dengan rumus:

$$\lambda = \frac{r(k-1)}{t-1}$$

Contoh Kasus Ban Mobil (Proyek Ban):

Kita menguji $t = 7$ jenis ban (A s.d G) dengan $b = 7$ sopir (blok). Tiap mobil hanya bisa memakai $k = 4$ ban sekaligus, dan tiap ban diulang $r = 4$ kali. Nilai $\lambda$ nya adalah:

$$\lambda = \frac{4 \times (4 - 1)}{7 - 1} = \frac{12}{6} = 2$$

Arti Praktis $\lambda = 2$: Jika kita mengambil dua jenis ban secara acak (misalnya ban A dan ban B), maka ban A dan B tersebut akan dipasang bersama-sama pada mobil yang sama oleh tepat 2 orang sopir di seluruh eksperimen. Hal yang sama berlaku untuk semua pasangan ban lainnya. Desain ini memastikan tidak ada pasangan perlakuan yang diistimewakan atau kurang dibandingkan.

Hubungan matematis antar parameter BIBD:

$$kb = rt$$

$$\lambda(t - 1) = r(k - 1)$$

Hipotesis:

$$H_0: \text{Semua perlakuan memiliki efek yang sama}$$

$$H_1: \text{Setidaknya ada satu perlakuan dengan efek berbeda}$$

Statistik Uji Durbin:

$$T = \frac{12(t-1)}{rt(k-1)(k+1)} \sum_{j=1}^t \left[ R_j - \frac{r(k+1)}{2} \right]^2$$

Dimana $R_j$ adalah jumlah peringkat perlakuan ke-$j$ yang diisi setelah dilakukan perangkingan *di dalam masing-masing blok* dari $1$ sampai $k$.

Kaidah Keputusan:

Tolak $H_0$ jika $T \ge \chi^2_{(1-\alpha, t-1)}$.

Kalkulator Uji Durbin (BIBD)

Pengantar Uji Goodness of Fit

Uji Goodness of Fit digunakan untuk memeriksa apakah fungsi distribusi kumulatif dari sampel data yang diamati ($S(x)$) sesuai dengan fungsi distribusi teoritis tertentu ($F^*(x)$), seperti Distribusi Seragam atau Normal.

1. Uji Kolmogorov-Smirnov (KS)

Mengukur jarak vertikal maksimum antara distribusi empiris sampel $S(x)$ dan distribusi teoritis hipotesis $F^*(x)$.

Statistik uji:

$$D = \max_{x} |S(x) - F^*(x)|$$

Secara operasional dihitung dengan mencari nilai terbesar di antara selisih berikut:

$$D_i^+ = \frac{i}{n} - F^*(x_{(i)})$$

$$D_i^- = F^*(x_{(i)}) - \frac{i-1}{n}$$

$$D = \max_{1 \le i \le n} (\max(D_i^+, D_i^-))$$

Dimana $x_{(i)}$ adalah data terurut.

2. Uji Cramér-von Mises (CvM)

Merupakan alternatif Kolmogorov-Smirnov yang lebih efisien karena menggunakan seluruh deviasi kuadrat di semua titik data, bukan hanya deviasi maksimum saja.

Statistik uji:

$$T = \frac{1}{12n} + \sum_{i=1}^n \left[ F^*(x_{(i)}) - \frac{2i-1}{2n} \right]^2$$

Dimana $n$ adalah ukuran sampel.

Kasus Uji Lilliefors (Normalitas):

Jika distribusi hipotesis adalah Distribusi Normal, namun parameter rata-rata ($\mu$) dan standar deviasi ($\sigma$) tidak diketahui dan harus diduga dari sampel ($\bar{X}$ dan $s$), maka uji KS standar menjadi tidak valid (karena terlalu konservatif). Uji penyesuaian ini disebut Uji Lilliefors, yang menggunakan tabel nilai kritis yang berbeda.

Koefisien Korelasi Non-Parametrik

Koefisien korelasi digunakan untuk mengukur kekuatan dan arah hubungan asosiasi antara dua peubah acak. Pada statistika non-parametrik, korelasi dihitung menggunakan peringkat (ranks) atau tabel kontingensi (kategori), sehingga bebas dari asumsi normalitas sebaran data.

1. Korelasi Peringkat Spearman ($r_s$ atau $\rho$)

Mengukur keeratan hubungan linier atau monotonik antara dua variabel acak berskala ordinal/interval. Peringkat dihitung secara terpisah untuk peubah acak $X$ dan $Y$.

Rumus Tanpa Ties (Peringkat Kembar):

$$r_s = 1 - \frac{6 \sum d_i^2}{n(n^2 - 1)}$$

Dimana $d_i = R(X_i) - R(Y_i)$ adalah selisih peringkat pasangan ke-$i$, dan $n$ adalah jumlah pasangan pengamatan.

Rumus Dengan Ties:

Jika ada angka kembar, kita berikan peringkat rata-rata (average ranks), dan $r_s$ dihitung sebagai koefisien korelasi Pearson dari peringkat tersebut:

$$r_s = \frac{\sum (R(X_i) - \bar{R}_x)(R(Y_i) - \bar{R}_y)}{\sqrt{\sum (R(X_i) - \bar{R}_x)^2 \sum (R(Y_i) - \bar{R}_y)^2}}$$

Pengujian Signifikansi (n > 30):

Menggunakan pendekatan normal standar dengan statistik uji:

$$Z = r_s \sqrt{n-1}$$

Tolak $H_0: \rho = 0$ jika $|Z| \ge z_{1-\alpha/2}$.

2. Korelasi Tau Kendall ($\tau$)

Mengukur kekuatan asosiasi berdasarkan jumlah pasangan pengamatan yang konkordan (searah) dan diskordan (berlawanan arah).

Misalkan terdapat pasangan $(X_i, Y_i)$ and $(X_j, Y_j)$:

  • Konkordan (P): Jika urutan arahnya sama, yaitu $(X_i \lt X_j \text{ dan } Y_i \lt Y_j)$ atau $(X_i \gt X_j \text{ dan } Y_i \gt Y_j)$.
  • Diskordan (Q): Jika urutan arahnya berlawanan, yaitu $(X_i \lt X_j \text{ dan } Y_i \gt Y_j)$ atau $(X_i \gt X_j \text{ dan } Y_i \lt Y_j)$.

Rumus Tanpa Ties:

$$\tau = \frac{P - Q}{n(n-1)/2}$$

Rumus Dengan Ties (Kendall's $\tau_b$):

$$\tau_b = \frac{P - Q}{\sqrt{(\frac{n(n-1)}{2} - T_x)(\frac{n(n-1)}{2} - T_y)}}$$

Dimana $T_x = \frac{1}{2} \sum t_i(t_i-1)$ dan $T_y = \frac{1}{2} \sum u_j(u_j-1)$ adalah faktor koreksi ties pada variabel $X$ dan $Y$.

3. Koefisien Kontingensi ($C_c$)

Digunakan untuk mengukur keeratan hubungan korelasi data kualitatif/nominal yang disajikan dalam bentuk tabel kontingensi baris $\times$ kolom ($r \times c$).

Uji Chi-Square ($\chi^2$):

$$\chi^2 = \sum_{i=1}^r \sum_{j=1}^c \frac{(O_{ij} - E_{ij})^2}{E_{ij}}$$

Dimana $O_{ij}$ adalah frekuensi pengamatan dan $E_{ij} = \frac{\text{Total Baris}_i \times \text{Total Kolom}_j}{N}$ adalah frekuensi harapan.

Koefisien Kontingensi ($C_c$):

$$C_c = \sqrt{\frac{\chi^2}{\chi^2 + N}}$$

Batas tertinggi nilai $C_c$ adalah $C_{max} = \sqrt{\frac{k-1}{k}}$ dimana $k = \min\{\text{jumlah baris } r, \text{jumlah kolom } c\}$.

Koefisien Korelasi Relatif:

$$r = \frac{C_c}{C_{max}}$$

Interpretasi Hubungan ($r$):

  • $r \lt 0.5$: Hubungan lemah
  • $0.5 \le r \lt 0.75$: Hubungan cukup kuat
  • $0.75 \le r \lt 0.90$: Hubungan kuat
  • $0.90 \le r \lt 1.0$: Hubungan sangat kuat
  • $r = 1.0$: Hubungan sempurna

Python (scipy & statsmodels) 
import numpy as np
import scipy.stats as stats

# 1. Korelasi Spearman & Kendall (Data Paired)
x = [48, 32, 40, 34, 30, 50, 26, 50, 22, 43]
y = [312, 164, 280, 196, 200, 288, 146, 361, 149, 252]

# Spearman
rho, p_val_sp = stats.spearmanr(x, y)
print(f"Spearman rho: {rho:.4f}, p-value: {p_val_sp:.4f}")

# Kendall's Tau
tau, p_val_kd = stats.kendalltau(x, y)
print(f"Kendall's tau: {tau:.4f}, p-value: {p_val_kd:.4f}")

# 2. Koefisien Kontingensi
# Tabel kontingensi
table = np.array([
    [82, 65, 12],
    [59, 112, 24],
    [37, 94, 42]
])

# Hitung Chi-Square
chi2, p_val, dof, expected = stats.chi2_contingency(table, correction=False)
n = table.sum()

# Hitung Cc
cc = np.sqrt(chi2 / (chi2 + n))
k = min(table.shape)
cc_max = np.sqrt((k - 1) / k)
r_rel = cc / cc_max

print(f"Chi2: {chi2:.4f}, p-value: {p_val:.4f}")
print(f"Cc: {cc:.4f}, Cc Max: {cc_max:.4f}, r relatif: {r_rel:.4f}")
